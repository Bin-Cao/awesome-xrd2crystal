## Überblick

Dieses Repository ist eine chronologisch kuratierte Sammlung von Papers, Code, Datensätzen, Benchmarks und Tools zu **XRD/PXRD zu Kristallstruktur**. Es umfasst Phasenidentifikation, Vorhersage von Kristallsystem, Raumgruppe und Gitterparametern, Mehrphasenzerlegung, Rietveld-Automatisierung, generative Strukturaufklärung sowie autonome und in-situ Analyse.

Die Bestimmung einer Struktur aus PXRD-Daten ist ein schlecht gestelltes inverses Problem: Viele Kandidatenstrukturen können ähnliche 1D-Diffraktogramme erzeugen, während experimentelle Profile durch instrumentelle Verbreiterung, Größeneffekte, Textur, Hintergrund, Fremdphasen und Peak-Überlappung beeinflusst werden.

| Richtung | Typisches Ziel |
|---|---|
| Phasenidentifikation | Bekannte Phasen aus PXRD-Mustern suchen oder klassifizieren |
| Symmetrie- und Gittervorhersage | Kristallsystem, Raumgruppe, Auslöschungsgruppe oder Gitterparameter vorhersagen |
| Mehrphasenzerlegung | Gemischte Muster trennen und Phasenanteile schätzen |
| Rietveld-Automatisierung | Verfeinerung, Parameterschätzung und globale Optimierung beschleunigen |
| Generative Strukturaufklärung | 3D-Kristalle, Atomkoordinaten, Gitter oder CIF aus PXRD erzeugen |
| Autonome/in-situ Analyse | Live-Diffraktionsdaten analysieren und experimentelle Feedbackschleifen schließen |

Um Zitationen, Links und Chronologie konsistent zu halten, wird die vollständige Liste in der englischen Hauptdatei gepflegt: [README.md](README.md#contents).
